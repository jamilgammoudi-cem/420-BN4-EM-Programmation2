Liste générique : Introduction

  • List<type> : La liste générique est une collection qui peut s’agrandir au besoin

    • Liste générique : Définir et instancier 

    // Définir et instancier une collection de nombres entiers
List<int> m_colNombres = new List<int>();
// Définir et instancier une collection de chaînes
string> m_colFruits = new List<string>();

    Liste générique : Ajouter des éléments

  • Add : Cette méthode permet d'ajouter un élément à la fin de la collection 
    m_colFruits.Add("Banane");

    • Liste générique : Accès aux éléments

  • Count : Permet d'obtenir le nombre d'éléments que contient la collection 
    int nombreDeFruits = m_colFruits.Count;
  • Obtenir un élément particulier de la collection en utilisant un index (position) 
    string premierFruit = m_colFruits[0]; // ici on obtient le premier élément de la collection
  • Modifier le contenu d'un élément particulier de la collection en utilisant un index (position) 
    // Remplacer le deuxième élément par "Pomme" 
m_colFruits[1] = "Pomme";

    • Liste générique : Retirer des éléments

  • RemoveAt : Permet de retirer un élément de la collection en utilisant son index 
    m_colFruits.RemoveAt(0); // ici on retire le premier élément de la collection
// Noter que tous les éléments suivant l'élément retiré seront déplacés d'une position

    • Liste générique : Recherche d'une élément

  • Contains : Permet de vérifier si une valeur est un des éléments de la collection ou non. 
    m_colFruits.Contains("Fraise"); // Cette méthode retourne true ou false selon que la valeur est un élément de la collection

    • Liste générique : Parcourir une liste avec une boucle

  • Exemple #1 : Parcourir une collection à l'aide d'une boucle foreach 
    // Parcourir la collection afin de compter le nombre de fruits de 6 lettres
int nbFruitsDe6Lettres = 0;
foreach (string fruitCourant in m_colFruits)
{
  if (fruitCourant.Length == 6)
  {
    nbFruitsDe6Lettres++;
  }
}

    • En parcourant une liste avec une boucle foreach, il est impossible de la modifier

  • Exemple #2 : Parcourir une collection à l'aide d'une boucle for 
    // Parcourir la collection afin de compter le nombre de fruits de 6 lettres
int nbFruitsDe6Lettres = 0;
for (int index = 0; index < m_colFruits.Count; index++)
{
  string fruitCourant = m_colFruits[index];
  if (fruitCourant.Length == 6)
  {
    nbFruitsDe6Lettres++;
  }
}

    • En parcourant une liste avec une boucle for afin de retirer certains éléments, il est conseillé de commencer à partir du dernier élément. Sinon, vous risquez de sauter quelques-uns !

    Instanciation avec new, référence

    1. Instanciation : création d'un objet avec new
    2. Accéder aux membres
      • propriétés : des valeurs appartenant à l'objet
      • méthodes : des actions ou traitements que l'objet peut effectuer
      • Constantes : lorsqu'une classe définit des constantes publiques, l'accès à ces constantes en dehors de la classe se fera toujours en utilisant le nom de la classe et non le nom de l'objet.
        exemple :  int valeur = MaClasse.NomConstante;
      • Méthodes statiques : lorsque la classe offre une ou plusieurs méthodes statiques, elles seront accessibles en utilisant le nom de la classe et non le nom de l'objet.
    3. Type valeur et type référence
        • Les variables qui stockent des valeurs en mémoire sont dites de type valeur (comme les types  int, char, decimal, bool,...)
        • Les variables qui stockent les adresses des objets en mémoire sont dites de type référence (comme les string, tableaux et classes)
        • Référence null : Si l'objet ne fait référence à aucun objet, sa valeur  null.  Un objet déclaré et non instancié a pour valeur null
    4. Variable locale et variable membre
      • Lorsqu'une variable est déclarée localement dans une méthode ou un bloc d'instructions, elle n'est visible est utilisable que dans la méthode ou le bloc où elle est déclarée. La variable est dite locale. Sa portée se limite à la méthode ou le bloc qui la déclare. Dès que l'exécution de la méthode ou du bloc est terminée, la variable n'existe plus.
      • Dans c# on peut déclarer des variables dont la portée est plus large (la classe). On dit que c'est une variable membre. Sa portée est toute la classe. Elle est utilisable dans toutes les méthodes de la classe. Pour les distinguer des variables locales, on adoptera le standard de les préfixer de m_ (m_prix par exemple). 
    5. Utilisation du débogueur pour visualiser les membres d'un objet
      • Lors de l'exécution, en mettant un point d'arrêt et en mettant le curseur sur l'objet, vous pouvez visualiser les valeurs des différentes propriétés de l'objet
      • Vous pouvez aussi voir la structure de la classe en faisant un clic droit sur le type de l'objet c'est à dire le nom de la classe et choisir "Atteindre la définition" dans le menu.

    Introduction aux classes

    1. Utilisation d'une classe : instancier un ou plusieurs objets à l'aide de l'opérateur new. 
      // Instanciation d'un objet facilitant la lecture d'un fichier texte
StreamReader objFichier = new StreamReader("UnFichierTexte.txt");

// Instanciation d'un objet permettant de générer des nombres aléatoires
Random m_objRandom = new Random();

// Instanciation d'une collection servant à contenir des objets de la classe Morceau
List<Morceau> m_colMorceaux = new List<Morceau>();
    2. class : mot clé servant à définir une classe
      • un identificateur significatif pour la nommer 
        public class CompteurHydro
{
    /// Définir un ou plusieurs champs
    /// Définir une ou plusieurs propriétés
    /// Définir un ou plusieurs constructeurs
    /// Définir une ou plusieurs méthodes
}
      • un ou plusieurs champs (variables membres privées)
      • une ou plusieurs propriétés pour donner accès aux champs
      • un ou plusieurs constructeurs servant à initialiser les champs
      • une ou plusieurs méthodes permettant de manipuler les champs
    3. Champ : une variable membre privée servant à stocker une information 
      private int m_consommationActuelle;
    4. Propriété : fournit l'accès à un champ en lecture et/ou écriture 
      public int ConsommationActuelle
{  
    // accès en lecture avec get
    get { return m_consommationActuelle; }
}
    5. Constructeur : initialise tous les champs (variables membres) 
      public CompteurHydro()
{
    m_consommationActuelle = 0;
}
    6. Méthode : permet d'effectuer un traitement sur un ou plusieurs champs 
      public void Consommer(int pQuantitéKwh)
{
    m_consommationActuelle += pQuantitéKwh;
}
