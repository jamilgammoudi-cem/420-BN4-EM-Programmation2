Liste générique : Introduction

  • List<type> : La liste générique est une collection qui peut s’agrandir au besoin

    • Liste générique : Définir et instancier 

    // Définir et instancier une collection de nombres entiers
List<int> m_colNombres = new List<int>();
// Définir et instancier une collection de chaînes
string> m_colFruits = new List<string>();

    Liste générique : Ajouter des éléments

  • Add : Cette méthode permet d'ajouter un élément à la fin de la collection 
    m_colFruits.Add("Banane");

    • Liste générique : Accès aux éléments

  • Count : Permet d'obtenir le nombre d'éléments que contient la collection 
    int nombreDeFruits = m_colFruits.Count;
  • Obtenir un élément particulier de la collection en utilisant un index (position) 
    string premierFruit = m_colFruits[0]; // ici on obtient le premier élément de la collection
  • Modifier le contenu d'un élément particulier de la collection en utilisant un index (position) 
    // Remplacer le deuxième élément par "Pomme" 
m_colFruits[1] = "Pomme";

    • Liste générique : Retirer des éléments

  • RemoveAt : Permet de retirer un élément de la collection en utilisant son index 
    m_colFruits.RemoveAt(0); // ici on retire le premier élément de la collection
// Noter que tous les éléments suivant l'élément retiré seront déplacés d'une position

    • Liste générique : Recherche d'une élément

  • Contains : Permet de vérifier si une valeur est un des éléments de la collection ou non. 
    m_colFruits.Contains("Fraise"); // Cette méthode retourne true ou false selon que la valeur est un élément de la collection

    • Liste générique : Parcourir une liste avec une boucle

  • Exemple #1 : Parcourir une collection à l'aide d'une boucle foreach 
    // Parcourir la collection afin de compter le nombre de fruits de 6 lettres
int nbFruitsDe6Lettres = 0;
foreach (string fruitCourant in m_colFruits)
{
  if (fruitCourant.Length == 6)
  {
    nbFruitsDe6Lettres++;
  }
}

    • En parcourant une liste avec une boucle foreach, il est impossible de la modifier

  • Exemple #2 : Parcourir une collection à l'aide d'une boucle for 
    // Parcourir la collection afin de compter le nombre de fruits de 6 lettres
int nbFruitsDe6Lettres = 0;
for (int index = 0; index < m_colFruits.Count; index++)
{
  string fruitCourant = m_colFruits[index];
  if (fruitCourant.Length == 6)
  {
    nbFruitsDe6Lettres++;
  }
}

    • En parcourant une liste avec une boucle for afin de retirer certains éléments, il est conseillé de commencer à partir du dernier élément. Sinon, vous risquez de sauter quelques-uns !
