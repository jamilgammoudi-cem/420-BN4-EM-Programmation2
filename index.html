Fichier texte

  1. Fichier texte en lecture StreamReader
    • Instanciation d'un fichier texte en lecture 
      //Instancier un fichier texte en lecture
StreamReader objFichier = new StreamReader("informations.txt");
    • La propriété EndOfStream : permet de vérifier si la fin du fichier est atteinte. Sa valeur est true lorsque la fin de fichier est atteinte. 
      StreamReader objFichier = new StreamReader("informations.txt");
while (!objFichier.EndOfStream)
{
    //instructions
}
    • La méthode ReadLine() : permet de lire une ligne de caractères du fichier et passe à la ligne suivante. Cette méthode retourne une string. 
      string chaine = "";
StreamReader objFichier = new StreamReader("informations.txt");
while (!objFichier.EndOfStream)
{
    chaine += objFichier.ReadLine() + " ";
}
    • La méthode Close() : permet de fermer le fichier 
      //Fermer le fichier
objFichier.Close();
  2. Fichier texte en écriture StreamWriter
    • Instanciation d'un fichier texte en écriture 
      //Instancier un fichier texte en écriture
StreamWriter objFichier = new StreamWriter("resultats.txt");
    • La méthode WriteLine() : permet d'écrire une ligne de caractères sur le fichier et passe à la ligne suivante. 
      //Écrit la valeur de la variable chaine dans le fichier
objFichier.WriteLine(chaine);
    • Il est important de fermer le fichier en écriture à la fin 
      //Fermer le fichier
objFichier.Close();
  3. Convertir une chaîne en nombre avec la méthode Parse(...)
    • La méthode Parse(...) : permet de convertir une chaîne de caractères en une valeur numérique entière. 
      // Dans cet exemple, elle va convertir la chaîne en une valeur numérique entière
int nombre = int.Parse(chaine);
    • La méthode Parse(...) : permet de convertir une chaîne de caractères en une valeur numérique décimale. 
      // Dans cet exemple, elle va convertir la chaîne en une valeur numérique décimale
decimal nombre = decimal.Parse(chaine);

Le contrôle ListBox

Utilisation

  1. Le control ListBox
    • Nous utiliseront le préfixe lst pour nommer les ListBox. Dans les exemples ci-dessous, lstFruits est un ListBox
    • La collections Items et la propriété Count 
      int nombreFruits= lstFruits.Items.Count;

      string fraise = lstFruits.Items[0].ToString();
    • La méthode Contains(...) 
      bool fruitExiste = lstFruits.Items.Contains("Kiwi");
    • La méthode Add(...): ajoute l'élément à la fin de la liste 
      lstFruits.Items.Add("Orange");
    • La méthode Remove(...): supprime l'élément de la liste 
      lstFruits.Items.Remove("Poire");
    • La méthode Clear(): pour vider la collection Items 
      lstFruits.Items.Clear();
    • La propriété SelectedIndex: Obtient ou définit l'index de l'élément sélectionné. La valeur de cette propriété est -1 si aucun élément n'est sélectionné. 
      int index = lstFruits.SelectedIndex;
lstFruits.SelectedIndex = 3;
    • La propriété SelectedItem :  Obtient ou définit la référence vers l'objet sélectionné dans la liste. 
      string unFruit = lstFruits.SelectedItem.ToString();
    • La propriété Text  
      string unFruit = lstFruits.Text;

Instanciation de classe avec new

  1. Instanciation : création d'un objet avec new
  2. Accéder aux membres
    • propriétés : des valeurs appartenant à l'objet
    • méthodes : des actions ou traitements que l'objet peut effectuer
    • Constantes : lorsqu'une classe définit des constantes publiques, l'accès à ces constantes en dehors de la classe se fera toujours en utilisant le nom de la classe et non le nom de l'objet.
      exemple :  int valeur = MaClasse.NomConstante;
    • Méthodes statiques : lorsque la classe offre une ou plusieurs méthodes statiques, elles seront accessibles en utilisant le nom de la classe et non le nom de l'objet.
  3. Type valeur et type référence
      • Les variables qui stockent des valeurs en mémoire sont dites de type valeur (comme les types  int, char, decimal, bool,...)
      • Les variables qui stockent les adresses des objets en mémoire sont dites de type référence (comme les string, tableaux et classes)
      • Référence null : Si l'objet ne fait référence à aucun objet, sa valeur  null.  Un objet déclaré et non instancié a pour valeur null
  4. Variable locale et variable membre
    • Lorsqu'une variable est déclarée localement dans une méthode ou un bloc d'instructions, elle n'est visible est utilisable que dans la méthode ou le bloc où elle est déclarée. La variable est dite locale. Sa portée se limite à la méthode ou le bloc qui la déclare. Dès que l'exécution de la méthode ou du bloc est terminée, la variable n'existe plus.
    • Dans c# on peut déclarer des variables dont la portée est plus large (la classe). On dit que c'est une variable membre. Sa portée est toute la classe. Elle est utilisable dans toutes les méthodes de la classe. Pour les distinguer des variables locales, on adoptera le standard de les préfixer de m_ (m_prix par exemple). 
  5. Utilisation du débogueur pour visualiser les membres d'un objet
    • Lors de l'exécution, en mettant un point d'arrêt et en mettant le curseur sur l'objet, vous pouvez visualiser les valeurs des différentes propriétés de l'objet
    • Vous pouvez aussi voir la structure de la classe en faisant un clic droit sur le type de l'objet c'est à dire le nom de la classe et choisir "Atteindre la définition" dans le menu.

Détails de la classe DisqueCompact
  • Il y a 2 surcharges du constructeur de la classe DisqueCompact
    • Le premier sans paramètre permet d'obtenir un disque avec un nombre de pistes et des durées de pistes qui sont aléatoires
    • Le deuxième permet de spécifier le type disque par exemple : DisqueCompact objDisque = new DisqueCompact(SorteDeDisque.Test1);
      • L'énumération SorteDeDisque offre 4 choix possibles, le premier est Normal pour obtenir un disque aléatoire les 3 autres sont fixes soit :
        • SorteDeDisque.Test1 est un disque de 2 pistes, la première de 59 secondes et la deuxième de 5:01
        • SorteDeDisque.Test2 est un disque de 8 pistes chacune d'une durée de 5 minutes
        • SorteDeDisque.Test3 est un disque de 12 pistes alternant séquentiellement entre 3:59 et 4:01
  • La propriété Durée fournit la durée total en secondes de toutes les pistes du disque
  • La propriété Genre fournit le genre de musique du disque, pour l'obtenir en chaîne faire Genre.ToString()
  • La propriété Pistes fournit un tableau contenant la durée de chacune des pistes
    • Pistes.Length fournit le nombre de pistes que contient le disque
    • Pour obtenir la durée d'une piste en particulier, il faut indexer la propriété Pistes
      • Pistes[0] c'est la durée en secondes de la première piste
      • Pistes[1] c'est la durée en secondes de la deuxième piste
  • Les méthodes FormatMMSS et FormatHHMMSS permettent respectivement de formater une durée en secondes 
    • par exemple : string duréeFormatée = DisqueCompact.FormatMMSS(duréeEnSecondesÀFormater);
